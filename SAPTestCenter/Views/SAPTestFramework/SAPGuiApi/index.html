How to get SAP GUI element

  • By Id Recommend

    Using Id is the fast and simplest way to get SAP Element

  • By Name

    Another way to identity a sap element is to use name,two parameters are needed, one is name the other is the sap element type

  • By Condition

    Sometime the id or name will change cause of new version or code change, but we can sure some property that won't change(may be height or width or other properties),and under this situation we can use one or more condition to identify the sap element
